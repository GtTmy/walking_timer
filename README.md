# walking_timer

インターバル速歩のためのタイマーアプリ for M5StickC

![m5](docs/images/pic.jpg)

## 操作方法

- 開始/一時停止
  - M5ボタン
- リセット
  - 右側ボタン

## インターバル速歩とは

早歩きとゆっくり歩きを３分ずつ交互に繰り返すこと。
